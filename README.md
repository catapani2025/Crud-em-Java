# Crud-em-Java
Projeto de back-end com Spring Boot e Java 21 que implementa APIs REST para gerenciar Categorias e Itens de inventário, usando PostgreSQL e Spring Data JPA. Oferece CRUD completo com validações, filtro por categoria, tratamento de erros (400, 404, etc.) e impede exclusão de categoria com itens vinculados. Código limpo e boas práticas.
